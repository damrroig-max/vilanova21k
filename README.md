# 🏃 Vilanova 21K — Media Maratón 2026

App de entrenamiento para David y Marta para preparar la Media Maratón de Vilanova el 15 de diciembre de 2026.

## Cómo instalar en el móvil (PWA)

Una vez desplegada en Vercel:

### iPhone (Safari)
1. Abre la URL en Safari
2. Pulsa el botón compartir (cuadrado con flecha)
3. Selecciona "Añadir a pantalla de inicio"
4. Pulsa "Añadir"

### Android (Chrome)
1. Abre la URL en Chrome
2. Pulsa los tres puntos (menú)
3. Selecciona "Añadir a pantalla de inicio"
4. Pulsa "Añadir"

## Desarrollo local

```bash
npm install
npm start
```

## Despliegue en Vercel

1. Sube este proyecto a GitHub
2. Conecta el repositorio en vercel.com
3. Vercel detecta automáticamente que es React
4. Pulsa "Deploy"
